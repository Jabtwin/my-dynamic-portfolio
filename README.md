# GitHub Dynamic Portfolio

A modern, dynamic portfolio application built with React, Vite, and Tailwind CSS. It fetches your GitHub data (repositories, profile information, etc.) dynamically and displays it in a beautiful, responsive UI.

## 📸 Screenshot

![Application Screenshot](images/screenshot.png)

## 🚀 Features

- **Dynamic GitHub Data**: Automatically fetches and displays your repositories and profile using the GitHub API.
- **Modern UI**: Styled with Tailwind CSS for a sleek, responsive, and accessible design.
- **Markdown Support**: Renders Markdown content flawlessly using `react-markdown`.
- **Fast & Lightweight**: Built on Vite for lightning-fast HMR and optimized builds.

## 🛠️ Tech Stack

- **Framework**: React 19 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React & React Icons
- **HTTP Client**: Axios

## 📦 Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Jabtwin/my-dynamic-portfolio.git
   ```

2. Navigate to the project directory:
   ```bash
   cd my-dynamic-portfolio
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Development Server

Start the development server:

```bash
npm run dev
```

Open `http://localhost:5173` to view the app in your browser.

### Building for Production

Create a production build:

```bash
npm run build
```
